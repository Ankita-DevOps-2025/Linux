# ALIAS in linux

# What is Alias?
A custom shortcut that represents:
    A longer command or
    A series of commands

# Alias Command Syntax
-> alias short_name="your command"
eg -> alias l="ls"    
alias myscript="cd /home/paul/tutprial/script"
alias rm="rm -i"

to see existing alisa -> alias -p

# use case of alias
Customizing Commands
Shortening Lengthy Commands
Navigating Directories (e.g., creating shortcuts for often-used paths)
Personalizing Commands

# for permanent alisa
update .bashrc and add your alias here 
then run -> source .bashrc

# make alias for globally/all user
update /etc/bashrc

# Benifit of alisa
Saves time and efforts
Increase productivity
Reduce risk of errors
Improved readability
